```shell script

    --from-literal=password=<password> \
    --type=kubernetes.io/basic-auth
```

### Nexus
```shell script

oc rollout pause dc nexus3
oc patch dc nexus3 --patch='{"spec": { "strategy": { "type": "Recreate" }}}'

oc rollout resume dc nexus3
```
